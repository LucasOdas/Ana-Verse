# 🌟 Ana-Verse 🌟

> Um universo digital criado especialmente para impressionar e divertir!

## ✨ Sobre o Projeto

**Ana-Verse** é uma aplicação web interativa e divertida, um universo pessoal criado com muita criatividade para você explorar, brincar e se divertir!

## 🎮 Funcionalidades

- **💌 Mensagens Especiais** - Receba mensagens personalizadas aleatórias
- **🎁 Gerador de Surpresa** - Desbloqueie surpresas e revelações especiais
- **🌌 Modo Galáxia** - Ative um modo visual especial com efeitos cósmicos
- **🎉 Festa Surpresa** - Lance confete e celebre!
- **📊 Estatísticas** - Acompanhe seus momentos especiais
- **✨ Efeitos Visuais** - Animações suaves e interativas

## 🎯 Como Usar

1. Abra o arquivo `index.html` no seu navegador
2. Clique nos botões para explorar as diferentes funcionalidades
3. Divirta-se e aproveite o Ana-Verse!

### 🔓 Dica Secreta
Tente pressionar **Ctrl + Shift + A** para desbloquear um easter egg especial! 🎊

## 💻 Tecnologias

- **HTML5** - Estrutura
- **CSS3** - Estilos e Animações
- **JavaScript** - Interatividade e Lógica

## 🎨 Recursos Especiais

O projeto utiliza um gradiente bonito de cores (roxo e rosa) com efeitos de glassmorphism para dar uma aparência moderna e elegante.
